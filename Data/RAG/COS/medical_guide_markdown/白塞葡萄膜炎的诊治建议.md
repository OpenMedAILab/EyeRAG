# 白塞葡萄膜炎的诊治建议

海峡两岸医药卫生交流协会风湿免疫病学专业委员会眼免疫学组
通信作者：郑文洁，中国医学科学院北京协和医院风湿免疫科，北京 100730，Email：zhengwj@pumch.cn；赵潺，中国医学科学院北京协和医院眼科，北京 100730，Email：zhaochan@pumch.cn

【摘要】 白塞葡萄膜炎（BU）是我国常见的致盲性较高的葡萄膜炎类型，其诊治面临诸多挑战，一方面，国内葡萄膜炎专科医师较为匮乏，大多数眼科医师对 的系统性用药不太熟悉；另一方面，风湿免疫科医生也难以掌握 的临床表现、诊断要点和评估方法。为提升我国 的诊疗水平，由葡萄膜炎和风湿免疫科专家组成的跨学科学组，在借鉴国内外诊治经验和指南的基础上撰写了本诊治建议，内容涵盖BU的临床表现、诊断与鉴别诊断、治疗及评估等，供广大眼科和风湿免疫病科医师参考。

【关键词】 葡萄膜炎； 白塞葡萄膜炎； 诊断； 治疗； 预后

白塞综合征（Behçet′s syndrome，BS）又称白塞病（Behçet′s disease，BD），是一种多系统受累的慢性、复发性自身免疫性/炎症性疾病，主要表现为反复发作的口腔溃疡、生殖器溃疡、眼部和皮肤损害，也可累及关节、消化道、心脏大血管、中枢神经系统等器官。BS好发于20\~30岁人群，男性更多见且症状更重。我国是BS高发地区之一，约 $3 5 \%$ 的BS患者有眼部受累［1］ ，可表现为葡萄膜炎、巩膜炎、角膜融解等，以白塞葡萄膜炎（Behçet′s uveitis，BU）最为常见［2］。BU是我国最常见的葡萄膜炎类型之一，也是 患者致残的最主要原因，预后较差、致盲率较高［2‑3］。为进一步提高该病的诊疗水平，在借鉴国内外诊治经验和指南的基础上，本学组撰写本诊断和治疗建议，供广大眼科和风湿免疫病科医师参考。

## 一、临床表现

BU通常表现为反复发作的后/全葡萄膜炎，也可表现为孤立性前葡萄膜炎或中间葡萄膜炎，但较为少见。最初可仅为单侧受累，但经过缓解‑复发的过程，多数患者发展为双侧受累［2，4］。

1.症状：BU后/全葡萄膜炎常表现为急性视力下降和眼前漂浮物，有些患者可在数小时内视力急剧下降；合并眼前节炎症时可有眼红、眼痛、畏光、流泪等眼部刺激症状。急性发作的常见诱因为劳累、熬夜、感冒、情绪激动，不治疗也有自发缓解的可能。部分患者在相对稳定期仍有发作性的眼前漂浮物，不影响视力或影响较小，数天至两周内可自行消失［5］ 。

2.体征：BU相关体征见表1。

3. 特殊检查：眼底荧光素血管造影（fundusfluorescein angiography，FFA）是评估视网膜血管炎的重要方法，BU急性发作时常可见视网膜静脉荧光素渗漏、着染，可出现无灌注区和出血遮挡引起的低荧光区域。弥漫毛细血管荧光素渗漏是 的特征性表现之一，与着染的视网膜静脉形成特征性的 蕨 树 叶 样 强 荧 光［2，5］。 光 学 相 干 断 层 扫 描（optical coherence tomography，OCT）可用于黄斑前膜、黄斑水肿、黄斑萎缩等黄斑病变的诊断和评估。眼部B超可检测玻璃体混浊等后节炎症表现，这在不能直接观察后节情况时尤有价值［2］ 。BU 急性发作时通常有显著的玻璃体混浊，在眼部B超中表现为玻璃体腔细密的点状中强回声。眼部B超还可检测到渗出性视网膜脱离、脉络膜增厚、视乳头水肿、重度黄斑水肿、玻璃体出血等可能出现于BU急性发作的表现。

## 二、诊断与鉴别诊断

分类诊断标准： 年白塞病国际研究小组（ISG）制定的BD诊断（分类）标准（ISGBD）曾广泛使用，灵敏度为 $8 5 \%$ ，特异度为 $9 6 \%$ 。但该标准将复发性口腔溃疡作为诊断的必要条件，加上以下任何两项方可诊断：复发性生殖器溃疡、眼部病变、皮肤损害或针刺反应阳性。该标准并不利于仅有口腔溃疡和典型眼炎的 的诊断［7］。2014 年白塞病国际诊断（分类）标准修订小组（ ）提出了修订后新标准。该诊断标准采用积分法，眼部病变、生殖器溃疡、口腔溃疡分别为2分，皮肤病变、神经系统表现、血管病变分别为1分，针刺反应阳性可以额外加1分，总得分≥4分者可诊断 $\mathrm { B S } ^ { [ 8 ] }$ 。2014 年的 ICBD 标准较 ISGBD标准显著提高了诊断的敏感性，同时保证了特异性（该标准灵敏度为 $9 4 . 8 \%$ ，特异度为 $9 0 . 5 \%$ ），目前已广泛用于临床（表 ）。

2.BU的诊断流程：葡萄膜炎是BS的重要诊断依据之一。BU一般表现为非肉芽肿性前葡萄膜炎，若是肉芽肿性的前葡萄膜炎则不符合；BU视网膜血管炎以静脉（血管鞘、视网膜出血、FFA荧光素渗漏、着染或无灌注区）和毛细血管受累（FFA可见视网膜血管蕨树叶样强荧光）为主，动脉受累并不显著，主要累及动脉的血管炎不符合BU；浸润性的视网膜炎性病灶符合BU表现，而坏死性视网膜炎性 病 灶 不 符 合 。 BU 的 诊 断 流 程 可 参 考Tugal‑Tutkun 等［9］ 发表的诊断算法。

3. 鉴别诊断：诊断BU还应除外其他葡萄膜炎类型，包括但不限于 Vogt‑小柳‑原田病（VKH）、人类白细胞抗原‑B27（HLA‑B27）相关葡萄膜炎、急性视网膜坏死、巨细胞病毒性视网膜炎、结节病相关葡萄膜炎、结核性葡萄膜炎、梅毒性葡萄膜炎、伪装综合征等（表3）［2］。

## 四、治疗原则及方案

### （一）治疗原则

BU后/全葡萄膜炎急性发作期的治疗原则是尽快控制炎症（一般需要全身性免疫抑制治疗），减轻因急性炎症发作引起的不可逆视功能损伤，同时积极防治各种急性期并发症。慢性期的治疗原则是长期控制病情、预防复发，防治慢性期并发症和药物不良反应［5，13‑14］ 。同时还需兼顾全身表现，个体化制定诊疗策略［11］。

### （二）全身用药

目前 的主要给药途径是全身用药，包括糖皮质激素、免疫抑制剂和生物制剂三大类。

1.糖皮质激素：糖皮质激素目前仍是治疗BU的主要药物之一，尤其是控制急性炎症。BU 病情较重且容易反复发作，一般需要给予长程口服糖皮质激素。在 BU 急性期，建议起始剂量泼尼松$1 . 0 \mathrm { m g / ( k g \cdot d ) }$ （或相当剂量），以达到快速控制炎症的目的。特别严重的BU急性发作可行激素冲击治疗（甲泼尼龙 $1 0 0 0 \mathrm { m g / d }$ ，连续3 d）。绝大多数患者对大剂量糖皮质激素治疗反应较好，可在治疗4周内病情明显好转的情况下开始减量；少数患者治疗反应较差，在确保诊断正确的情况下，可酌情观察更长时间，或尽快加用其他二线药物。减量过程应逐步放缓，若在减量过程中炎症复发，糖皮质激素剂量可提高到减量前水平或更高，然后再次缓慢减量至之前复发的剂量，并适当延长维持时间。全身应用糖皮质激素的减量应根据病情、患者治疗反应、合并用药等情况个性化制定，同时要关注血压、糖脂代谢、眼压和骨密度等糖皮质激素长期治疗的不良反应以及病毒性肝炎和结核的复燃［15‑17］。

2.免疫抑制剂：由于长期应用糖皮质激素不良反应较大，需联合免疫抑制剂以降低其长期应用的累积剂量。对于 BU，目前证据较多的免疫抑制剂是环孢素 A［CsA，常用剂量 $3 { \sim } 5 ~ \mathrm { m g / ( k g \cdot d ) } .$ ］和硫唑嘌呤［AZA，常用剂量 $2 { \sim } 2 . 5 \ \mathrm { m g / ( k g \cdot d ) } \ ]$ ，对严重病例两者可联合使用。随机对照临床试验显示，CsA和AZA均可保护视力并减少BU复发。其他免疫抑制剂，比如他克莫司（TAC，常用剂量 $2 { \sim } 3 ~ \mathrm { m g / d } )$ 、霉酚酸酯（MMF，常用剂量 $1 . 0 { \sim } 1 . 5 ~ \mathrm { g } / \mathrm { d } .$ ）等也有治疗BU 后/全葡萄膜炎的报道。甲氨蝶呤（MTX，每周常用剂量 $1 0 { \sim } 1 5 ~ \mathrm { m g }$ ）的疗效较弱，但耐受性较好，可用于较轻的BU或作为稳定期用药［5］。

免疫抑制剂的应用还要考虑到安全性。钙调磷酸酶抑制剂 $( \mathrm { C s A } , \mathrm { T A C } )$ ）需警惕高血压、肾损伤，因其神经毒性不适用于神经 患者；使用 前建议评估硫嘌呤甲基转移酶（TPMT）基因多态性，警惕严重骨髓抑制；MTX 需监测血常规和肝功能［5，14］ 。

3.生物制剂：人基因重组 $\alpha$ ‑干扰素 $\mathrm { \Delta } \operatorname { \left[ F N \alpha \right] }$ ）：目前用于治疗BU的 $\mathrm { I F N } \alpha$ 主要是 $\mathrm { I F N } \alpha 2 \mathrm { a }$ ，也有应用$\mathrm { I F N } \alpha 2 \mathrm { b }$ 的报道，但缓解率略低于 $\mathrm { I F N } \alpha 2 \mathrm { a }$ ；研究显示$\mathrm { I F N } \alpha 2 \mathrm { a }$ 对难治性复发性BU有效，既可抑制BU眼部急性期炎症，也能作为慢性期的维持治疗，在停药后仍能维持长时间的病情稳定［13，18‑20］。

国际上尚无统一的 $\mathrm { I F N } \alpha 2 \mathrm { a }$ 治疗BU的方案。$\mathrm { I F N } \alpha 2 \mathrm { a }$ 的给药途径为皮下注射，起始剂量一般为每日 $3 { \sim } 6 { \times } 1 0 ^ { 6 } \mathrm { ~ U ~ }$ ，一般不超过 $9 \times 1 0 ^ { 6 }$ ，一般持续 $2 \sim$ 8周，维持剂量一般为 $3 { \times } 1 0 ^ { 6 }$ U隔日1次或每周3次。$\mathrm { I F N } \alpha$ 耐受性好，其不良反应通常与剂量相关，以流感样症状最常见，其次为轻度白细胞 血小板减低，肝酶升高，脱发等。有诱发自身抗体阳性及甲状腺疾病的报道，故建议治疗期间监测自身抗体和甲状腺功能。此外，需要注意其增加抑郁及自杀的风险。和抗肿瘤坏死因子 $\mathrm { \Delta T N F } \alpha$ ）制剂比较，IFNα无潜在结核再激活的风险，而且由于它本身可以用于治疗乙型肝炎，可用于合并乙型肝炎病毒（hepatitisB virus，HBV）感染的患者［13，20］ 。

抗 $\mathrm { T N F } \alpha$ 单抗：常用治疗 的 单抗的剂量和用法如下：英夫利西单抗（IFX）， $5 ~ \mathrm { m g / k g }$ 静脉滴注，第 、、周各 次，以后每 周 次；阿达木单抗（ADA），首剂 $8 0 ~ \mathrm { m g }$ 皮下注射，1 周后每 2 周$4 0 \ \mathrm { m g }$ ，体重 ${ < } 3 0 \mathrm { ~ k g }$ 的儿童剂量减半［5，21］ 。 研究显示，抗 $\mathrm { T N F } \alpha$ 单抗应用 1年以上，且维持 3\~6个月缓解下可逐渐调整剂量或延长给药间隔［22‑23］。

使用前应进行全身基础疾病的评估，除外严重基础疾病（心力衰竭）、感染（特别是反复感染病史、结核病、活动性乙型肝炎）、肿瘤（尤其是淋巴瘤）和中枢神经系统脱髓鞘疾病。诊断活动性结核感染者，不推荐应用 $\mathrm { T N F } \alpha$ 抑制剂，或在专科医生指导下完成活动性结核病治疗后再考虑使用；合并潜伏性结核感染者，应给予预防性抗结核治疗至少4周后可开始使用 $\mathrm { T N F } \alpha$ 抑制剂［24］ 。 对于乙型肝炎表面抗原（HBsAg）阳性的慢性HBV感染者，需在使用$\mathrm { T N F } \alpha$ 抑制剂前预防性抗病毒治疗（恩替卡韦或替诺福韦酯），并持续至治疗结束后；HBsAg阴性、乙型肝炎核心抗体（HBcAb）阳性患者，若HBV‑DNA阳性，需要进行预防性治疗；如HBV‑DNA阴性，可启用 抑制剂治疗，但用药期间需每 个月监 测 丙 氨 酸 转 氨 酶（ALT）水 平 、HBV‑DNA 和，一旦 或 转为阳性，应立即启动抗病毒治疗［25］。

### （三）局部用药

眼局部治疗主要是糖皮质激素滴眼液（眼膏）、眼周注射和玻璃体腔注射三大类。

1.局部滴（涂）眼：常用糖皮质激素滴眼液主要有 $0 . 1 \%$ 地塞米松磷酸钠 $, 1 \%$ 醋酸泼尼松龙、 $0 . 1 \%$ 氟米龙及 $0 . 5 \%$ 氯替泼诺等。前两者抗炎强度较强，可用于眼前节炎症急性发作，后两种抗炎效果相对弱一些，对眼压的影响也较小，适用于急性炎症控制后的降阶治疗或激素性高眼压（或高危）患者［17］ 。

一般在活动期用较高频率的糖皮质激素滴眼液快速控制前节炎症，随后逐渐减量直至停药。前房炎症发作或复发时，应根据其严重程度选择合适的滴眼频率。治疗过程中，根据炎症消退情况逐渐减量，最初可降低滴眼频率，维持期可转换成抗炎作用弱的糖皮质激素滴眼液［17］。

糖皮质激素滴眼液的主要不良反应是眼压升高和白内障形成，使用期间应密切关注眼压变化。

2.眼周注射：眼周注射与滴眼液相比有眼内浓度高、作用时间长、眼后段抗炎作用强等特点。常用的眼周注射方式包括球结膜下注射、 囊下注射、球周注射和球后注射等。注射药物包括地塞米松磷酸钠、甲泼尼龙琥珀酸钠、曲安奈德注射液等。各种药物的吸收时间随剂型不同而异，地塞米松吸收时间最短，可 $1 { \sim } 2 \mathrm { ~ d ~ }$ 注射1次；甲泼尼龙略长，可 注射 次；而曲安奈德为混悬制剂， $2 { \sim } 3 \mathrm { ~ d }$ 起效，吸收时间最长，可 个月注射 次［17，26］。

球结膜下注射一般注射体积为 $0 . 5 { \sim } 1 . 0 ~ \mathrm { m l }$ ，地塞米松 $2 . 5 { \sim } 5 . 0 ~ \mathrm { m g }$ 或曲安奈德 $1 0 { \sim } 2 0 ~ \mathrm { m g }$ 。结膜下注射可同时达到较高的前房和玻璃体腔药物浓度，但引起眼压升高的危险也高于球侧和球后注射。Tenon 囊下注射包括前 Tenon 囊和后 Tenon 囊下注射。前Tenon囊下注射的适应证等同于球结膜下注射，后 囊下注射主要用于治疗眼后段的炎症。球周注射是把药物注射到眶隔后肌锥外，球后注射则是指注射入肌锥内，两种注射方法均可以达到较高的眼后段药物浓度。一般注射体积为 $0 . 5 \sim$ $1 . 0 \mathrm { m l }$ ，地塞米松 $2 . 5 { \sim } 5 . 0 ~ \mathrm { m g }$ 、曲安奈德 $2 0 { \sim } 4 0 ~ \mathrm { m g }$ 或甲泼尼龙 $2 0 { \sim } 4 0 ~ \mathrm { m g }$ 。主要用于眼后段炎症或黄斑水肿［17］ 。

3.玻璃体腔注射：玻璃体腔注射可达到很高的眼内浓度，具有很好的抗炎效果。常用的药物是曲安奈德（在我国目前是超适应证应用），注射剂量为$\mathrm { 1 { \sim } 2 ~ m g }$ ，一次注射起效8\~16周，主要用于单眼顽固性黄斑水肿、严重全/后葡萄膜炎或双眼较重的一侧。玻璃体腔糖皮质激素缓释剂主要成分是地塞米松或氟轻松，疗效维持时间与药品剂型、规格等因素相关，植入后可长时间（6\~36个月）在玻璃体腔缓慢释放所含地塞米松或氟轻松。全身不良反应可忽略，但白内障、眼压升高等局部不良反应不

容忽视［15］ 。

### （四）手术治疗

手术治疗主要针对 的局部并发症，包括并发性白内障、药物难以控制的继发性青光眼等。并发性白内障应在眼部炎症稳定至少3个月才可实施手术，如存在大范围虹膜粘连或前节炎症反复发作病史，可在围手术期适当增加免疫抑制药物剂量。对于难治性青光眼，传统的小梁切除术必须联合术中应用抗代谢药，否则长期失败率较高。微创青光眼手术（MIGS）目前也逐步应用于葡萄膜炎继发性青光眼，但报道还比较少。为防止手术失败，抗青光眼围手术期也需要积极地进行免疫抑制治疗。

### （五）治疗方案选择［27‑29］

孤立性前葡萄膜炎急性发作以局部治疗为主，但有预后不良因素（青年、男性及发病年龄早）者可考虑全身免疫抑制剂治疗（如AZA，AZA禁忌下可考虑CsA或MTX）。对于BU后/全葡萄膜炎，如未危及视力，应给予全身糖皮质激素联合免疫抑制剂，如AZA和（或）CsA，反复复发者可联合抗TNFα单抗或 $\mathrm { I F N } \alpha$ 治疗；针对急性威胁视力的葡萄膜炎，抗TNFα单抗或 $\mathrm { I F N } \alpha$ 可作为一线用药，部分患者可考虑大剂量甲泼尼龙冲击治疗。顽固性黄斑水肿者可换用托珠单抗。单眼恶化者，在系统治疗基础上，可考虑眼周或玻璃体腔糖皮质激素注射作为辅助治疗。

## 五、预后

BU在病程最初5年内常反复发作，致盲率高，预后较差。总体而言，男性患者较女性患者预后差，发病年龄越小预后越差，眼后节受累较眼前节受累者预后差，早期及时正确治疗对改善患者预后极为重要。

综上，BU是我国最常见的葡萄膜炎类型之一，也是BS患者致残的最主要原因，临床预后较差、致盲率较高。 多表现为反复发作的后 全葡萄膜炎，起病较急，以显著玻璃体炎、视网膜血管炎和视网膜浸润灶为特征性表现。诊断需符合2014年白塞病国际研究小组制定的BS分类和诊断标准，并除外其他葡萄膜炎类型。 的治疗目标是尽快控制急性炎症，防止不可逆视功能损伤；长期控制病情、预防复发，防治并发症和药物不良反应。同时还需兼顾全身表现，个体化制定诊疗策略。本建议是专家基于现有证据提出的，不具备法律效力，会在新证据不断出现的情况下继续更新。
