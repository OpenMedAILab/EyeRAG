随着抗逆转录病毒治疗（antiretroviral therapy，ART)的广泛应用，人类免疫缺陷病毒(human im-munodeficiency virus, HIV)感染者（people living withHIV，PLWH)的生存期显著延长，慢性并发症管理已成为PLWH防控的新挑战。其中，眼部机会性感染作为晚期的主要致残因素，其防治体系构建直接影响患者生存质量与公共卫生负担。截至2023 年，全球累计PLWH逾4000万"，约 $5 \% \sim 2 5 \%$ 的患者会出现失明2]。自上世纪开始提倡对PLWH进行眼科筛查以来[3]，系统性眼科评估显著提升了PLWH的眼病检出率(筛查组 $1 2 . 8 \%$ ，非筛查组 $1 . 5 \%$ )[4。我国农村地区整体防控效果欠佳，PLWH人数呈上升趋势[5]，同时受医疗资源不足及筛查体系薄弱影响，PLWH人群的眼部疾病未能获及时干预[6。亟待完善基层眼科筛查网络。

巨细胞病毒性视网膜炎(cytomegalovirus retini-tis,CMVR)是PLWH免疫功能严重缺陷、进入艾滋病（acquired immune deficiency syndrome, AIDS）期的界定性疾病之一， $\mathrm { C D 4 ^ { + } T }$ 淋巴细胞 ${ < } 1 0 0$ 个 $\mathbf { \nabla } / \mu \mathrm { L }$ 的HIV感染者中，CMVR发病率达 $3 0 \% { \sim } 4 0 \% ^ { \mathrm { ( 2 ) } }$ 。根据我国大城市住院病例数据统计，我国在进入ART治疗时代后，CMVR发病率在 $7 . 6 \% \sim 9 . 0 \% ^ { [ 7 . 8 ] }$ 。尽管ART可部分恢复免疫功能，但延迟诊断或治疗仍会导致视网膜不可逆损伤，严重影响视觉相关生活质量。

基于上述背景，中国罕见病联盟/北京罕见病诊疗与保障学会眼科罕见病专业委员会组织多学科专家，结合国内外循证证据与临床实践经验，通过反复会议讨论，围绕CMVR的筛查、诊断、治疗及随访全流程制定本规范。通过优化筛查技术路径、细化分层治疗策略及完善随访管理体系，旨在为各级医疗机构提供科学、可操作的实践指导，降低CMVR致盲率，助力“健康中国2030”目标实现。

## 一、巨细胞病毒性视网膜炎筛查

### (一)筛查对象

1.CMV血症：血CMV-DNA阳性者；

2.有其他系统CMV病者，如CMV肺炎、CMV肠炎或CMV脑炎；

3.有眼部症状者，如视物模糊，眼前黑影等；

4.未启动ART或者ART失败；

5.HIV-RNA ${ \mathrm { > } } 1 { \times } 1 0 ^ { 6 }$ 拷贝 $/ \mathrm { m L }$ ;

6.最低 $\mathrm { C D 4 ^ { + } T }$ 淋巴细胞数 $< 2 0 0$ 个 $/ \mu \mathrm { L }$ 者[9-10]。

建议对满足上述任一条件的CMVR高危人群进行眼底筛查。

### (二)筛查时间和频率

1.初次检查:在确诊HIV感染后尽早进行全面眼科检查；

2.定期随访（初次检查无CMVR者）:（1)CD4+T 淋巴细胞计数 $- 5 0$ 个 $\mathbf { \nabla } / \mu \mathrm { L }$ 者，每1个月眼底筛查； (2)CD4+T淋巴细胞计数50\~100个 $/ \mu \mathrm { L }$ 者，每3个 月眼底筛查； $( 3 ) \mathrm { C D 4 ^ { + } T }$ 淋巴细胞计数 $1 0 0 { \sim } 2 0 0$ 个 $/ \mu \mathrm { L }$ 者，每6个月眼底筛查；(4)直到 $\mathrm { C D 4 ^ { + } T }$ 淋巴 细胞计数 ${ > } 2 0 0$ 个 $/ \mu \mathrm { L } ^ { [ 1 1 ] }$

急症筛查：患者出现眼部症状时，需立即眼科就诊。

推荐意见一：对CMVR高危人群应完善眼底检查以筛查CMVR，对于初筛阴性者应根据 $\mathrm { C D 4 ^ { + } T }$ 淋巴细胞水平定期复筛。

### （三)筛查手段和方法

1.检眼镜：包括直接检眼镜或间接检眼镜。通过直接眼底镜可以观察视网膜中央区域（黄斑区）和视盘等结构，但视野较小，仅能观察视网膜的部分区域，且需要与患者近距离接触；借助双目间接检眼镜或裂隙灯前置间接眼底镜可观察视网膜周边区域，视野范围大，适合评估广泛的视网膜病变筛查。眼科医师通过检眼镜进行散瞳眼底检查是传统的诊断CMVR的关键手段，还适用于住院卧床患者的眼底病筛查。

2.眼底照相：CMVR的眼底临床特征明显，筛查的关键在于尽量全眼底范围查看，避免漏诊。应行普通视野眼底照相和(超）广角眼底照相。(1)普通视野眼底照相：可用于筛查严重威胁视力的后极部CMVR 病灶。一次成像范围 $2 5 ^ { \circ } { \sim } 6 0 ^ { \circ }$ 。患者瞳孔直径 $\geqslant 3 . 0 \ \mathrm { m m }$ 时可免散瞳，易于操作；当瞳孔直径 $< 3 . 0$ $\mathrm { m m }$ 时，予药物散瞳，以拍摄高质量图像。为尽量全面记录眼底情况，每眼捕获9个标准化眼底图片，软件将9个图像合成拼图，可覆盖 $8 5 ^ { \circ } { \sim } 1 0 5 ^ { \circ }$ 眼底范围[12-14]。拍摄顺序为后极部、颞侧、颞上、上方、鼻上、鼻侧、鼻下、下方和颞下。图像可用于远程会诊和随访对比。拍摄地点可以在眼科或传染病专科门诊或病房。眼科医师可以通过远程会诊普通视野眼底照片拼图诊断CMVR（图1），其灵敏度为 $3 0 . 2 \% \sim$ $100 \%$ ,特异性为 $9 9 . 1 \% { \sim } 9 9 . 9 \%$ ,这与部分病灶范围小、位于视网膜周边部有关，因此获得周边部成像的眼底照相技术将有助于提高检查的准确性[12-13,15]。由于采集范围有限，单纯采用普通视野眼底照相无法明确周边部视网膜有无CMVR病灶或完整了解

CMVR病灶的范围。（2)超广角眼底照相：眼球正位一次成像可达到赤道前部至锯齿缘范围的技术，称为超广角眼底成像技术。以Optos 超广角激光扫描眼底照相技术为例，无需散瞳，瞳孔直径 $\geqslant 2 \ \mathrm { m m }$ 即可单次成像范围达到 $2 0 0 ^ { \circ }$ ，覆盖眼底 $80 \%$ 以上范围，还能通过眼位引导功能扩展到 $2 2 0 ^ { \circ } { \sim } 2 4 0 ^ { \circ }$ 。拍摄时尽量避免眼睑或睫毛的遮挡，必要时使用开睑器。双眼分别在正位，上方、下方、鼻侧和颞侧眼位拍摄图像[16。使用超广角眼底照相单视野成像筛查CMVR的灵敏度 $9 6 . 3 \%$ ,特异性为 $100 \%$ ；与传统散瞳前置镜眼底检查相比，筛查结果的一致性很好[7]。与传统的普通视野眼底照相的拼图相比，超广角眼底像捕获的视网膜面积增加了 $4 8 . 3 \%$ ,CMVR病灶面积增加了 $4 0 . 0 \% ^ { [ 1 8 ] }$ 另外，对于有CMVR病灶者还应详细评估病灶的位置，用于预后评估和科学研究：1区指距离视盘一个视盘直径或距离黄斑中心凹两个视盘直径的区域，2区指1区以外至涡静脉壶腹间的区域，3区指2区以外至锯齿缘的区域范围[1%。超广角眼底照相填补了普通视野眼底照相的不足，能高效拍摄全视网膜范围，判断有无CMVR病灶，并完整了解病灶累及的范围。

推荐意见二:通过全视网膜范围的检眼镜检查和/或眼底照相技术筛查CMVR，眼底照相推荐优先采用超广角眼底照相。

## 二、巨细胞病毒性视网膜炎诊断依据

CMVR主要根据眼底表现和患者全身情况进行诊断，对眼内液进行CMV-DNA检测可辅助于诊断和鉴别诊断。

1.眼部症状：可无任何眼部症状，仅在眼底筛查时发现；或有视力下降，局部视野缺损，眼前黑影飘动感。

2.眼部体征：(1)眼前节正常；或有尘状或羊脂状角膜后沉着物，前房闪辉；玻璃体透明或有不同程度的炎性混浊。（2)眼底表现： $\textcircled{1}$ 颗粒型：呈黄白色片状或簇状病灶，活动性边缘呈颗粒状，进展缓慢，出血不明显； $\textcircled{2}$ 出血水肿型：俗称“奶酪番茄酱”样改变，大片黄白色视网膜水肿坏死灶和出血灶，位于视网膜后极部或周边部，进展迅速； $\textcircled{3}$ 视神经视网膜炎：黄白色病变和出血灶累及视神经及附近视网膜； $\textcircled{4}$ 霜样树枝状视网膜血管炎：血管壁变白，呈霜样树枝，往往伴有上述其他类型视网膜病灶（图2）。以上不同类型视网膜病灶可在同一眼底同时出现，甚至可能伴有陈旧的视网膜萎缩灶。(3)病灶分布：CMVR病灶在眼底分布无特异性，局灶或象限性分布，或散在分布，病灶距黄斑距离1500$\mu \mathrm { m }$ 以内称为中央型病变，其他为周围型病变。病程较长者可见病灶中心视网膜萎缩灶，无出血水肿，但其边缘呈活动性、白色簇状颗粒样病灶。

3.相干光断层扫描（optical coherence tomogra-phy,OCT）：在CMVR中可用于与其他眼底病鉴别、明确黄斑区是否被CMVR病灶侵犯、有无并发黄斑水肿或前膜、病灶区有无视网膜浅脱离、辨别活动性病灶（视网膜增厚，层次不清）或陈旧性病灶（视网膜萎缩或机化)等，有助于评估视力预后。

4.眼内液核酸检测：怀疑CMVR的眼内液标本以采用房水即可。操作前抗生素滴眼液点眼。可采用平卧位或于裂隙灯显微镜下坐位采集患者的前房水标本 $0 . 0 5 { \sim } 0 . 1 \ \mathrm { m L }$ ，注入无菌无酶可密封容器内，常温下72小时内送检。操作结束后继续抗生素

滴眼液点眼[20]。

眼内液核酸检测不足之处：有创；假阴性可能，即CMVR病灶很小（如初发期病灶<1PD)时，房水CMVR核酸检测可能为阴性[21]；治疗过程中CMVR核酸拷贝数的波动对病情发展的指导价值尚不明确，不建议以该检测作为常规的随访检查项目。因此，眼内液核酸检测为诊断CMVR的辅助手段，在无条件进行检测时，同样可进行CMVR的诊断和治疗。

眼内液中的CMV-DNA阳性并不代表有感染视网膜的“活病毒”。从病原学诊断的角度，活动性CMVR的诊断金标准是病毒分离培养，但由于该检测手段灵敏性较低，对实验条件要求高，临床应用较少[9。

以CMVR眼部症状首诊于眼科的PLWH易被误诊，由于患者及眼科医生对全身病情不清，可能被误诊为糖尿病视网膜病变、视网膜分支静脉阻塞、缺血性视神经病变、白塞病等[2]。对于HIV感染诊断明确的患者，考虑CMVR时还应与其他眼底病相鉴别，如HIV相关视网膜病变[23]、进展性外层视网膜坏死[24]、梅毒性葡萄膜炎[25]、眼结核[26-28]、药物的眼部不良反应[29]等。

推荐意见三：由经验丰富的眼科医生对CMVR进行诊断，眼内液CMV-DNA的检测有助于CMVR的诊断和鉴别诊断。

## 三、巨细胞病毒性视网膜炎治疗及随访管理

对CMVR不主张提前预防用药，眼底筛查明确有CMVR时开始积极治疗。首先考虑全身用药抗CMV治疗，一方面为控制患眼病情，预防对侧眼患病；另一方面也是为了控制CMV全身病情，提高患者生存率；必要时联合玻璃体注药。

1.全身治疗：分为诱导期（14\~21天)和维持期治疗（3\~6个月)[""]。目前国内推荐用药有更昔洛韦、缬更昔洛韦和膦甲酸钠。诱导期用药：更昔洛韦$5 . 0 { \sim } 7 . 5 ~ \mathrm { m g / k g / }$ 次，Q12H,静脉滴注；或膦甲酸钠60$\mathrm { m g / k g / }$ 次，Q8H,静脉滴注；或麟甲酸钠 $9 0 ~ \mathrm { m g / k g / }$ 次，Q12H,静脉滴注;或缬更昔洛韦 $9 0 0 ~ \mathrm { m g / }$ 次，Q12H，口服。维持期用药：更昔洛韦 $5 ~ \mathrm { m g / k g / }$ 次，QD，静脉滴注；或缬更昔洛韦 $9 0 0 ~ \mathrm { m g } , \mathrm { Q D }$ ，口服；或更昔洛韦$1 0 0 0 ~ \mathrm { m g }$ ,TID，口服；或膦甲酸钠 $9 0 { \sim } 1 2 0 ~ \mathrm { m g / k g / }$ 次，QD，静脉滴注。针对危重症眼病患者或怀疑耐药时，可改用膦甲酸钠或联合使用更昔洛韦和膦甲酸钠。

在我国当前医疗条件下，CMVR患者诱导期治疗多在医院住院完成，住院期间每周复查眼底，记录眼底病灶发展情况；出院后口服更昔洛韦维持期治疗，每月复查眼底。当患者眼底活动性病灶消退，且 $\mathrm { C D 4 ^ { + } T }$ 淋巴细胞数 ${ > } 1 0 0$ 个 $/ \mu \mathrm { L }$ 并持续3\~6个月时可停用抗CMV药。在抗CMV治疗开始2周内尽快启动ART。

注意事项：PLWH在AIDS期可能在用药前就表现为白细胞减少，或因为合并肿瘤疾病需要进行化疗，同时上述抗CMV药物同时具有骨髓抑制、肾毒性或电解质紊乱等不良反应，治疗过程中需要多学科协作（感染科和眼科），根据患者免疫功能状态、全身情况及眼部病变动态调整治疗方案。

2.玻璃体注药：玻璃体内注射抗病毒药物可在短时间内提高眼内药物浓度，更高效控制病情。适用于危及视力的中央型CMVR 患者或全身用药有禁忌者(如严重骨髓抑制、妊娠早期患者)9。

操作流程与眼科其他疾病的玻璃体注药相同。玻璃体内注射更昔洛韦 $( 2 { \sim } 4 \ \mathrm { m g } / 0 . 0 5 { \sim } 0 . 1 \ \mathrm { m L }$ ，每周1\~2次）,或者膦甲酸钠 $( 2 . 4 ~ \mathrm { m g / 0 . 1 ~ m L }$ ，每周1\~2次)。由于每只患眼的CMVR病情严重程度各异、患者全身病情各异，临床上对于玻璃体注药的用药剂量、注射频率以及次数尚无严格统一标准，主要以眼底病变程度为参考。不建议参考眼内液病毒拷贝数，因为眼内液中的CMV-DNA并不是可继续感染视网膜的“活病毒”。另外，玻璃体注药前后应进行详细眼科检查，警惕操作或药物（更昔洛韦）相关并发症，如高眼压、玻璃体积血、黄斑水肿、药物结晶样改变或视网膜色素改变[30-32]。

推荐意见四：不建议为预防CMVR而对PLWH进行预防性抗CMV治疗，推荐在多学科协作下对确诊的CMVR进行全身抗CMV治疗，对危及视力的中央型CMVR患者或全身用药有禁忌的患者采用玻璃体给药。

推荐意见五：不建议以监测眼内液病毒拷贝数作为评估CMVR病情转归的常规手段。

推荐意见六：当活动性CMVR 病灶完全消退，且 $\mathrm { C D 4 ^ { + } T }$ 淋巴细胞数 ${ > } 1 0 0$ 个 $/ \mu \mathrm { L }$ 并持续3\~6个月时可停用抗CMV治疗。

3.预后随访：CMVR患者在维持期治疗期间或病情稳定数年后，仍可能会出现患眼病情复发或视力再次下降，如免疫重建性葡萄膜炎或视网膜脱离的可能。

(1)免疫重建性葡萄膜炎(immune recovery u-veitis，IRU)：出现在ART启动后数月到数年不等。表现为前房炎症、玻璃体炎、视盘/黄斑水肿、黄斑前膜、并发性白内障、玻璃体黄斑牵引综合征、增生性玻璃体视网膜病变甚至视网膜新生血管，导致患者视力预后更差。IRU轻者仅有眼前节和玻璃体炎性反应，易被忽视。CMVR 患者IRU发病率 $5 \%$ \~$7 1 \% ^ { \left[ 3 3 \right] }$ ，既往CMVR病灶范围越大的患者越容易出现IRU。对PLWH进行积极眼底筛查、早期诊断和治疗CMVR有利于减少IRU的发生。另外，若在启动ART前未对已存在的CMVR病灶进行筛查和治疗，后期在免疫功能重建后会出现CMVR病情加重，炎性反应加重，即巨细胞病毒免疫重建性视网膜炎（cytomegalovirus-immune recovery retinitis )[34]

推荐意见七：重视CMVR患者病程中出现的威胁到视力的免疫重建性葡萄膜炎反应，谨慎使用局部糖皮质激素或联合抗 CMV 治疗控制炎症与病毒活化风险。

## 四、小结

综上所述，CMVR是PLWH在AIDS期最常见的致盲性眼病，积极的眼底筛查是防盲的关键，超广角眼底照相为优选手段。CMVR的诊断以临床诊断为主，眼内液检测为辅。多学科协作下全身和/或玻璃体给药治疗。终生随访和治疗CMVR 相关并发症。
