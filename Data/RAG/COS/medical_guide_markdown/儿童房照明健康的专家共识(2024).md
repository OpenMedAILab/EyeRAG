# 儿童房照明健康的专家共识（2024）

中国医学装备协会眼科专业委员会视觉功能与视觉工效评估学组

【摘要】 人类在出生后视觉系统仍然处于发育状态，适宜的照明环境有利于儿童视觉系统的发育和身心健康。当前，教室和活动室照明环境对于防控儿童近视眼的发生已有广泛的认知。但是，在儿童居家照明环境方面却缺乏相应的健康指导。针对儿童房照明健康问题，中国医学装备协会眼科专业委员会视觉功能与视觉工效评估学组对儿童居家生活、学习及娱乐环境的照明健康问题进行研究讨论，在儿童房照明布局、照明控制及照明光源等方面形成专家共识。旨在引起儿童房照明健康的社会关注，规范儿童居家照明环境，助力我国儿童近视眼防控。

联合国儿童权利公约将儿童界定为 18岁以下的任何人。儿童视觉系统的发育与光环境之间存在复杂的关系。既往，已有研究结果表明光环境对儿童近视眼的发生和发展存在明确的影响［14］ 。当前，我国儿童近视眼的发生率仍居高不下。其原因可能与家庭照明情况相关。通常，多数家庭在儿童房装修时更关注于实用与美观，缺少对儿童视觉发育和照明健康环境方面的考虑。为此，中国医学装备协会眼科专业委员会视觉功能与视觉工效评估学组汇集了眼科学、视觉生理学、照明设计与光源制造等领域专家的意见，对儿童房照明布局、照明控制及照明光源等达成如下共识。

儿童房照明健康的相关因素

一、照明光源

1．自然光源：太阳光是地球表面的主要自然光源，人类适应了地面的自然光环境，自然光被认为是“理想的光源”。自然光的光谱是连续的，无尖锐的高峰和低谷亦无闪烁，显色指数为 $1 0 0 ~ \mathrm { R a }$ a。每天上午和下午特定一段时间内的自然光适于阅读，而早晚光线强度过低或正午光线直射情况下则不适于阅读，自然光亦非均为“健康的光”。此外，太阳光受地球四季回归线及气象条件等多因素的影响。因此，儿童房的照明目前还无法实现只采用自然光源作为照明光源。

2．人造光源：现代社会的主要人造光源包括白炽灯、荧光灯及发光二极管（lightemittingdiode，LED）光源。白炽灯发光效率低且耗能大，家庭照明领域已经不再应用。荧光灯发光效率高且寿命长，但是其光谱连续性差且存在闪烁问题。目前，LED灯正在逐步替代荧光灯和白炽灯。普通 LED灯由于蓝色光谱成分占比高且中长波长的光谱成分占比低，存在引起视疲劳的危险因素。近年来，市场上出现的全光谱LED灯具备低蓝光、低频闪及高显色指数且接近自然光的光谱特点，在儿童房的照明应用中具有其独特的优势。

二、照明布局设计

分区照明是儿童房照明设计的一项重要内容。根据儿童活动内容，儿童房的功能区域至少包括休息区、学习区及活动区。各功能区域的照明需求不同，应针对不同区域的特定需求分布光源。为了提高学习的专注度，学习桌应配备台灯。台灯放置于左前方，亮度要稍高于环境亮度。不能以台灯代替室内照明，即在近距离工作时需要同时打开台灯和房间灯光。台灯照明时间可以根据儿童年龄进行定时控制，避免长时间近距离用眼。此外，合理的照明设计还可有效避免或减少眩光和阴影的影响，提高室内光照均匀度，减少室内照明环境对儿童产生的不适度和视觉疲劳。与筒灯和射灯相比，面板灯的照度均匀度更高。而无主灯设计或见光不见灯设计则有利于达到更高的照度均匀度。但是无主灯设计方案中，若要提高工作面的照度水平，则需要配备更高功率的光源，照明的耗能也随之增大。因此，若从环保的角度出发，无主灯设计还需进一步突破光源耗能的技术问题。

三、照明控制系统的设计

室内自然采光条件受自然因素的影响大，普通单一的照明控制系统不能随着自然采光条件的变化而变化。照明光源参数的自适应控制技术可随外界光线的变化而对室内照明进行调节，保证儿童房内各功能区域工作任务面恒定的照度和周围环境照度的均匀度。智能控制系统也可实现随活动内容调节照明的需求。理想的照明控制系统还可以从生物节律、情绪调节角度出发，对儿童房的照明进行节律性地调节。如清晨调节为高相关色温灯光照明促进觉醒，午休或夜间时采用低相关色温灯光照明以稳定情绪，改善睡眠。近年来，随着物联网技术的进步，智慧照明已经开始用于公共照明和路灯管理领域，以“人因”为出发点的智慧照明控制系统将是儿童房照明控制系统的发展趋势。

儿童房照明健康相关因素与近视眼的相关性

一、照明光源与近视眼的相关性

1．照明光源的类型：有学者注意到美国儿童近视眼发生率存在2个高峰，即上个世纪90年代末和2010年，近视眼发生率分别由 $2 5 \%$ 增长至 $40 \%$ 和由 $40 \%$ 增长至 $50 \%$ 。该时期美国学生的学习时长和户外活动时间未见明显变化，长时间使用电脑是在 2000年以后，智能手机和平板电脑的普及则是在2010年以后。可见，使用电脑并未额外增加美国学生近视眼的风险。而照明光源技术的迭代似乎与近视眼发生率存在一定的关联。上个世纪 70年代，荧光灯开始在美国替代白炽灯；2000年后，LED灯开始普及并与近视眼患病率的两个增长高峰重合［3］ 。有学者发现在采用荧光灯照明的房间睡眠，2岁前幼儿屈光参差的发生率增加，荧光灯发出的光有诱导散光发生的倾向［4］ 。另有研究结果表明，使用LED台灯的学生较使用荧光灯或白炽灯学生近视眼的发展程度更深，提示使用 LED台灯可能是加重近视眼的危险因素［5］ 。目前，市场在售的各类“护眼灯”其光源参数不同，能否达到护眼效果需要谨慎对待。

2．照明光源的光谱波长：光线经正视眼屈光系统折射后，不同波长的光聚焦在视网膜附近不同的点位上可形成纵向色差和垂轴色差。其中，纵向色差是指不同波长的光束折射后所形成的焦平面位于视网膜前后，处于离焦状态的不同位置上。若中波长的焦平面恰好落于光感受器平面，短波长蓝光焦平面则位于视网膜光感受器之前，而长波长红光的焦平面则位于光感受器之后。纵向色差是调节眼球发育的信号之一。不同动物视网膜光感受器类型、敏感光谱和数目不同，不同波长的光环境作用对屈光发育的影响不同。短波长的蓝光可诱导鸡、鼠及猪等动物的屈光状态向远视方向发展，长波长的红光则诱导其屈光状态向近视眼方向发展。而树鼩和恒河猴两种动物则呈现相反的结果。有调查资料表明，与色觉正常的儿童相比色觉缺陷儿童的近视眼发生率更低、近视眼进展更慢且轴向增长率更小［6］ 。推测可能因色觉异常者对特定波长光的敏感度降低故而对眼球生长的影响较小，这一结果与早期单色光实验的预测结果一致。不同光谱的光源与近视眼发生之间存在复杂的关系，人类视网膜在进化过程中已适应自然光的光谱特性。因此，建议采用全光谱光源照明使照明光源尽量接近自然光的光谱成分，从而避免荧光灯或普通 LED光源对视觉发育的不利影响。

3．照明光源的光照强度：人眼可适应 $1 0 . 6 \mathrm { \ c d / m } ^ { 2 }$ ～$1 0 6 ~ \mathrm { c d / m } ^ { 2 }$ 的光强度，其范围广泛故既能在正午工作又能在微弱的星光下活动。室外光照强度一般在 $1 0 3 \sim 1 0 4 ~ \mathrm { { l u x } }$ 之间，而室内人工照明光照强度一般为 $1 0 2 ~ \mathrm { { l u x } }$ 甚至更低与自然光相差甚远。多项动物实验的研究结果和流行病学调查结果表明，更强的照明环境和室外活动均有利于近视眼防控［79］ 。低照度环境可诱导实验动物鸡产生近视眼，而提高照度可以预防和控制近视眼的发生与发展［10］ 。低强度照明环境不是导致灵长类实验动物近视眼发生的直接因素，但是高强度照明环境可延缓实验动物猴形态剥夺性近视眼的发展，提示户外活动过程中的照明强度提高可能有延缓近视眼发展的作用［11］ 。有学者发现阅读面照度越低越容易导致代偿性眼轴延长且近视眼者较正视者更明显，并认为高强度照明环境延缓近视眼的发展可能与视网膜内多巴胺分泌的调节有关［9］ 。照度越低调节距离越大，而实际眼调节越小调节落后就越大。视觉分辨阈感受性符合韦伯定律，眼在过强的光环境下需要更多的调节和快速视觉适应才能发现细微的差别。因此，室外强光环境或距离采光附近的窗口区更易引起视疲劳，不利于长时间的阅读。反之，在远离窗口的采光区或朝暮时段及阴天情况下光照强度过弱也不适宜阅读。

4．照明光源的色温：当光源的色品与某一温度下黑体的色品相同时，该黑体的绝对温度为此光源的相关色温，简称色温。大于 $5 3 0 0 ~ \mathrm { K }$ 者称为冷色温，小于3300K者称为暖色温， $3 3 0 0 \sim 5 3 0 0 \mathrm { ~ K ~ }$ 者称为中间色温。色温 $6 5 0 0 ~ \mathrm { K }$ 且显色指数96与北半球正午阳光环境相似，定为标准光源，记为D65。采用此标准光源可以保证阴雨天在室内观测物品的颜色近似于在太阳光下观测的效果。色温对心理情绪影响比较大，冷色温环境使人精神活跃，警戒程度高；而暖色温环境容易使人放松，帮助入睡。低色温光环境类似于上午或下午的自然光环境，可能更有利于近距离用眼。有学者分别采用$3 0 0 0 \textrm { K } . 4 0 0 0 \textrm { K }$ 及 $5 0 0 0 ~ \mathrm { K }$ 三种色温的 LED灯光作为环境照明的主光源观察色温对猕猴眼轴发育的影响，结果发现低色温光能有效延缓猕猴眼轴的生长速度且这种影响在整个观测期间持续存在［12］ 。 $3 0 0 0 \mathrm { ~ K ~ }$ 色温的 LED灯光包含短波长成分较低而长波长成分较高； $5 0 0 0 ~ \mathrm { K }$ 色温的 LED光源包含短波长成分较高而长波长成分较低。在选择光源时，一般宜选择全光谱低蓝光的 LED光源。

二、照明的布局设计与近视眼的相关性

照度均匀度是照明设计中的关键性指标之一，即最小照度与平均照度的比值。户外自然环境下，局部区域的照度均匀度为1，感受不到光线强弱的变化。而在室内，由于距离采光窗口或距离光源位置的不同以及室内家具或物品的遮挡，使得室内光线分布并不均匀。当光照不均匀时，人眼便会被迫不断调节瞳孔和晶状体的状态来适应明暗变化。这种频繁的调节不仅加重了视疲劳，还可能导致视觉系统的长期紧张，从而增加近视眼的风险。照明布局设计不仅要考虑照明的实用性和艺术性，还要针对儿童年龄和活动类型选择适宜的光源种类和给光方式并注意墙面或家具对光的折射影响，避免不舒适眩光或不均匀照明。原则是在满足基本需求的前提下，照明布局设计还应考虑视觉的舒适性以不易产生视觉疲劳为宜。此外，在条件允许时，照明布局设计还应充分考虑照明对非视觉效应的影响。理想的照明布局设计应进行实地观测给出更精准的设计和光源安放位置。

三、照明控制设计与近视眼的相关性

照明与人的昼夜节律密切相关，环境照明的改变可影响人的睡眠觉醒周期和生理代谢活动，进而对情绪和情感等精神心理活动产生影响。人类和动物多巴胺递质和褪黑素的释放、眼轴长度的变化以及相关基因的表达也存在昼夜节律变化且与近视眼的发生相关［13］ 。有流行病学调查结果表明睡眠时间与近视眼的发生相关，夜间睡眠时间越晚越易发生近视眼［14］ 。以“人因”为出发点对环境照明进行智能控制以适应生理节律，将有宜于抑制近视眼的发生与发展。

儿童房照明健康的推荐方案

一、照明布局设计原则

测量室内空间高度和面积，根据功能区的需求，个性化设计面板灯布局和面板内部光源布局，避免出现眩光。

二、照明控制设计原则

采用自适应光调节和昼夜节律设置，实现引导睡眠和促进觉醒的智能照明环境控制。

三、照明光源种类推荐

推荐使用光谱特征、显色指数及色温接近自然光的全光谱 LED光源。

儿童房健康照明的评估方法

学习区、娱乐区和手工区的照明指标检测平面为任务平面，休息区的照明指标检测平面为床面。照度、色温及显色指数等照明指标可通过照度计进行测量。其中，照度均匀度采用最小照度值与平均照度值的比值进行测算。全光谱光源的品质可以通过光谱仪进行测量，以光谱范围、不同波长所占成分比（光谱连续性 CS值）及一般显色指数和特殊显色指数进行评估。照明领域，根据自主感光型神经节细胞和似昼夜节律的研究结果提出了黑视素照度指标。黑视素照度指标是以 D65标准日光光源的照度为对照，用于量化光源对黑视素影响，主要针对的是节律照明与似昼夜节律的关系。黑视素照度指标可以作为儿童房照明健康光源设计的参考，也可作为儿童睡眠节律和心理情绪的一种调节手段。此外，由于光对人生理和心理的影响较大程度上取决于眼对光强弱的感知。因此，照明领域还提出了室内生理等效照度指标。此指标是用于量化儿童眼位高度垂直面上的光照强度。建议儿童房的夜间生理等效照度控制在 $5 0 ~ \mathrm { l u x }$ 以下。
