【Abstract】 For vitreous or retinal choroidal diseases of unknown origin, diagnostic vitrectomy can be performed for auxiliary diagnosis. With the development of laboratory testing technology and minimally invasive vitrectomy in recent years, diagnostic vitrectomy for biopsy has been widely used in clinical practice. To standardize the clinical application of diagnostic vitrectomy for biopsy, the Ocular Oncology Committee of China Anti‑Cancer Association, Ophthalmological Society of China Medicine Education Association has organized relevant experts and developed this consensus after repeated discussions on the role of diagnostic vitrectomy for biopsy, the key points of operation, specimen delivery, test selection, and complications.

【Key words】 Vitrectomy; Biopsy; Consensus

不明原因的眼内炎、玻璃体或视网膜脉络膜恶性疾病不仅可致盲，还可能威胁生命，往往需要依靠活体检查进行诊断，以便尽快确定病变性质，并给予有效治疗。传统的经巩膜细针穿刺玻璃体活体检查术操作简单，但获取的标本量少，而且由于患眼的玻璃体液化程度不确定，非直视下穿刺不一定能成功抽取到玻璃体。自20世纪60年代开展玻璃体切除手术以来，随着技术水平不断提高，该手术从高难度、创伤大、恢复慢逐渐发展成为微创手术，具有创口小、无需缝合、手术方式简化、手术时间缩短、术后愈合快等优势，致使诊断性玻璃体切除手术逐步取代了传统的经巩膜细针穿刺玻璃体活体检查术，成为目前诊断感染性眼内炎、眼内恶性疾病的重要方法。

为保证诊断性玻璃体切除手术的安全性和有效性，中国抗癌协会眼肿瘤专业委员会和中国医药教育协会眼科专业委员会汇集国内相关专业专家，基于国内外应用现状，结合临床经验和研究结果，经过反复讨论，针对诊断性玻璃体切除手术的规范化应用提出共识性意见，以期为临床开展工作提供指导和参考。

## 一、诊断性玻璃体切除手术适应证

在诊断性玻璃体切除手术中，可更清晰观察眼内病灶并获取玻璃体、视网膜或脉络膜组织标本，根据各项实验室检查结果明确病因，从而提高诊断的准确率。诊断性玻璃体切除手术前应详细询问病史，并进行眼科检查和全身检查，初步判断病情。若根据全身和局部检查结果不能明确诊断，且病情无法控制，视力或生命可能受到严重威胁，可考虑行诊断性玻璃体切除手术［1］。

在临床中诊断性玻璃体切除手术多用于眼内感染、伪装综合征或眼内占位性病变的明确诊断。

### （一）眼内感染

部分感染性葡萄膜炎具有典型的眼底表现或全身病史，如病毒性视网膜炎、眼弓蛔虫病、眼弓形体病或眼梅毒等，根据典型的临床表现或房水检测结果不难诊断［2‑4］ 。当临床表现不典型、可疑多重感染或需要与伪装综合征进行鉴别诊断，且房水标本量不足或眼底病情尚未累及房水时，可考虑行诊断性玻璃体切除手术获取有效标本，以助于提高诊断的准确性。由细菌或真菌引起的玻璃体和（或）房水感染，称为眼内炎或感染性眼内炎，其致病菌种类繁多，眼底表现缺乏特异性，破坏性大，可在症状出现数小时或数天内导致感染眼不可逆转失明，可考虑行诊断性玻璃体切除手术获取玻璃体标本，以明确诊断并给予有效治疗；对于继发于角膜感染的眼内炎，前房的炎性反应较重，通常房水检测即可明确病因。大多数感染性眼内炎均需要行诊断性玻璃体切除手术，玻璃体标本的阳性率明显高于玻璃体穿刺液或房水［5］。

对于不明原因的葡萄膜炎，当临床考虑为非感染性葡萄膜炎时，不建议行诊断性玻璃体切除手术，因为内眼操作可能加重病情，且并不一定能明确病因。

### （二）伪装综合征

部分恶性肿瘤可表现为眼部玻璃体炎，包括眼内淋巴瘤（最常见）、白血病、转移性癌等。这类疾病病情重，诊断困难，可考虑行诊断性玻璃体切除手术，以便尽快明确诊断。尤其在高度怀疑眼内淋巴瘤时，须行诊断性玻璃体切除手术以获取玻璃体标本，或可同时行脉络膜视网膜活体检查［6‑7］。

### （三）眼内占位性病变

位于虹膜、睫状体、脉络膜或视网膜的大部分眼内占位性病变或肿瘤可通过非手术方法进行诊断。对于少数临床表现不典型的眼内占位性病变，需要进行组织病理学诊断，可行诊断性玻璃体切除手术以获取病灶组织标本［8］。

对特殊情况下的眼内肿瘤，尽管临床已明确诊断，也建议行诊断性玻璃体切除手术。当患者或某些治疗方案要求提供组织病理学诊断，可考虑行诊断性玻璃体切除手术。在国外葡萄膜黑色素瘤组织活体检查已成为敷贴放射治疗手术的常规操作，基于活体检查的基因表达谱检测有助于进一步判断肿瘤转移的风险并指导治疗［9‑11］。

## 二、诊断性玻璃体切除手术的操作要点

操作原则为获取足量、有效的组织标本，控制手术并发症，避免加重原发病。20G、23G、25G甚至 微创玻璃体切除手术系统均可用于行诊断性玻璃体切除手术。为提高诊断的准确率，应在抗感染治疗前或抗生素类药物使用前行诊断性玻璃体切除手术。为了避免糖皮质激素对淋巴细胞的裂解作用，对于怀疑淋巴瘤者诊断性玻璃体切除手术前1个月应停用糖皮质激素。

### （一）单纯诊断性玻璃体切除手术

多采用经睫状体平坦部单通道玻璃体切除手术。

在瞳孔区直视下，玻璃体切割头经单通道巩膜穿刺口进入玻璃体腔。

2. 术者一边轻压眼球以保持眼压，一边切吸玻璃体腔中央部未稀释的玻璃体液，切割频率设置在2 500 次/min以上。对高度怀疑眼内淋巴瘤等眼内肿瘤者，在获取玻璃体液细胞学检测标本时，应将切割频率设置在 $6 0 0 { \sim } 8 0 0$ 次 $/ \mathrm { { m i n } } ^ { [ 1 2 ] }$ 。

助手用注射器连接切割头的抽吸管道，在术者切割的同时缓慢回抽约 $1 { \sim } 2 ~ \mathrm { m l }$ ；退出玻璃体切割头，残留于切割头抽吸管道内的未稀释玻璃体液也应回抽入注射器。

4. 经巩膜穿刺口缓慢注入等量的平衡盐溶液，以维持眼压。

5. 术后在巩膜顶压下详细检查周边视网膜［13］。

单纯诊断性玻璃体切除手术操作简单，经济安全；获取中轴部玻璃体腔未稀释的玻璃体液，标本量有限。适用于初次行诊断性玻璃体切除手术获取玻璃体标本。

### （二）诊断性全玻璃体切除手术

采用经睫状体平坦部三通道玻璃体切除手术。

1. 完成三通道巩膜切口后，不打开灌注通道。

2. 在玻璃体混浊明显处或视网膜病灶附近，术者一边轻压眼球以保持眼压，一边切吸玻璃体腔未稀释的玻璃体液。

助手用注射器连接切割头的抽吸管道，在术者切割的同时缓慢回抽约 $1 { \sim } 2 ~ \mathrm { m l }$ 。

4. 退出玻璃体切割头，残留于切割头抽吸管道内的未稀释玻璃体液也应回抽入注射器。

5. 打开灌注通道，诱导玻璃体后脱离，削剪玻璃体后皮质至玻璃体基底部，此时可继续使用注射器获取稀释的玻璃体。

必要时进行眼内激光或气液交换等特殊眼内操作；手术结束前详细检查眼底，尤其周边视网膜［13‑15］ 。

诊断性全玻璃体切除手术可获取更多、更有效的组织标本，提高检出率［1］ ，必要时可同时处理眼底并发症。但是，手术难度相对较大，术中屈光介质可能存在不同程度混浊，尤其在玻璃体周边部，故术中应谨慎操作，尽量避免造成医源性视网膜裂孔［16］ 。

由于炎症细胞、淋巴瘤细胞多聚集在玻璃体皮质，故建议在保证安全的前提下，应对怀疑感染性眼内炎或眼内淋巴瘤者采用诊断性全玻璃体切除手术，更利于获取有效的组织标本［14］ 。

### （三）视网膜脉络膜活体检查

采用经睫状体平坦部三通道玻璃体切除手术。必要时可在吊顶灯辅助下行双手同时操作［17］。

尽量完全切除玻璃体，尤其病灶附近的玻璃体，以免在视网膜脉络膜病灶区行活体检查时出现视网膜或玻璃体嵌顿。

2. 病灶组织标本获取位置选择：（1）在视网膜脉络膜活跃病灶与正常视网膜的边界处获取组织标本，此处细胞复制最为活跃；（2）避开病灶中富含血管或纤维化区域；（3）尽可能位于眼底上方，便于术后填充气体；（4）尽可能选择鼻侧相对无血管区域，以减少术中出血；（5）尽可能位于赤道后，便于微创手术器械操作。

3. 获取组织标本前提高眼内灌注压。

4. 可采用眼内穿刺、眼内电凝切割或玻璃体切割头高负压低切速等方式获取病灶组织标本。

必要时电凝止血，眼内激光或冷冻固定视网膜切口边缘。

### （四）其他

诊断性玻璃体切除手术可结合实际情况，获取眼内多种组织标本，如视网膜前膜［16，18］ ，视网膜下

液［19‑20］ 等。对于眼前房占位性病变，可选择两通道角膜切口诊断性玻璃体切除手术，用玻璃体切割头获取病变组织［21］。

## 三、诊断性玻璃体切除手术标本的送检及实验室检测

诊断性玻璃体切除手术可获取3类玻璃体标本，即未稀释玻璃体液、稀释的玻璃体和手术积液盒液体。此外，还可获取视网膜下积液、视网膜或脉络膜的固体组织。标本的规范化送检流程和检测方法选择，可决定诊断性玻璃体切除手术的诊断准确率。

### （一）标本送检

标本送检的基本原则是尽快送检，规范送检。

1. 标本送检时，临床医师应同时提供与疾病相关的详细临床资料。

标本的处理方式取决于送检时间，即从获取标本到实验室检测之间的时间间隔。若送检时间在1小时内，标本无需进行固定处理；若送检时间可能超过 $^ { 1 \mathrm { ~ h ~ } }$ ，则需要将标本放入培养基（如牛血清清蛋白）或温和的固定液中。固定液可保护标本中的细胞形态、免疫活性和 DNA 结构。传统的固定液包括福尔马林和戊二醛，此外还有CytoLyt固定液和羟乙基哌嗪乙磺酸 谷氨酸介导的具有保护作用的有机溶剂［22］。

3. 需 要 进 行 聚 合 酶 链 反 应（polymerase chainreaction，PCR）、抗体检测、微生物培养的标本需要在冷藏或冷冻环境下送检。可疑眼内感染者，标本应放置在无菌瓶或离心管中，防止污染。固体组织标本应迅速置于固定液中送检，必要时需要标明标本的正确方位。

### （二）实验室检测

根据诊断性玻璃体切除手术前的临床诊断和具体的标本量，针对性选择检测方法，必要时联合多种检测方法。

病原微生物检测：对怀疑或需要排除感染性病因者，应进行病原微生物检测。PCR是基于病原体核酸的检测，包括实时定量PCR和多重PCR等。PCR的阳性率高，操作快捷，可选用稀释的玻璃体，临床中最常用于检测眼内病毒感染［23］ 。酶联免疫吸附试验或免疫印迹技术可检测标本的病原微生物抗体，适用于检测眼弓形体感染［24］ 、眼弓蛔虫感染［25］ ，也可联合检测病毒性葡萄膜炎，同时与血液中的抗体滴度进行比较（Goldmann‑Witmer系数）。微生物培养和涂片检测首选未稀释玻璃体液，但培养周期长，培养结果易受术前用药影响；涂片检测常无法确定具体微生物的类型和种属。涂片检测和微生物培养常联合 用于检测感染性眼内炎。宏 基 因 组 学 第 二 代 测 序（metagenomics nextgeneration sequencing，mNGS）可直接对标本中的核酸进行高通量测序，与数据库比对分析后，判断标本中包含的病原微生物（病毒、细菌、真菌、寄生虫）。对于新发、罕见、疑难感染性疾病，以及免疫缺陷者可能合并的多重感染，mNGS可作为一线检测方法。

细胞病理学检查：诊断性玻璃体切除手术中获取的标本多数为液体，适宜做细胞病理学检查。为保持细胞形态，送检的液体标本最好不加固定液、不稀释，迅速送至实验室进行离心，然后将离心后的沉淀物直接涂片，风干后行常规染色，在光镜下进行细胞形态学观察。由于细胞病理学检查获取的信息有限，其诊断效能受限，对病理科医师的诊断经验依赖性较大。

流式细胞术可利用液体标本检测不同细胞表面的标记物或抗体，整体流程简便，可辅助诊断眼内淋巴瘤。实时定量 检测免疫球蛋白基因重排，判断有无异常增生的淋巴细胞，也可辅助诊断眼内淋巴瘤。

组织病理学检查：诊断性玻璃体切除手术中获取的视网膜脉络膜固体标本，适宜进行组织病理学检查。将标本常规置于固定液中进行组织固定，然后进行石蜡包埋、切片，行常规苏木精 伊红染色和免疫组织化学染色等，在光镜下观察。组织病理学检查结果为诊断的金标准。

4. 细胞因子检测：采用酶免疫分析法、免疫印迹技术或分子生物学方法检测玻璃体液中某些细胞因子水平，可辅助诊断。目前临床最常用、最具有临床指导价值的检测指标是白细胞介素（interleukin，IL）10 与 IL‑6 的比值，大于 1 高度提示细胞来源玻璃体视网膜淋巴瘤。除此以外，检测炎性反应因子或细胞因子对葡萄膜炎的诊断价值尚不明确，不建议临床推广［12］，但可用于科学研究，如自身免疫性葡萄膜炎研究［26］。

值得注意的是，尽管各种检测方法日新月异，但其结果仅能提供参考。临床医师在解读检测结果时应考虑到可能的假阳性、假阴性、实验室污染或非致病性微生物等无临床意义的结果；疾病不同阶段的检测结果也可能存在差异［24］。因此，最终的诊断应是临床医师对尽量全面的病史、查体和各种实验室检测结果以及疗效观察进行综合判断后得到的结果。

此外，由于诊断性玻璃体切除手术获得的标本宝贵，对病理科、检验科的检测技术要求较高，因此其实验室要有相应的技术条件和资质，且临床医师、病理科医师和实验室技师之间需要密切沟通协作。

## 四、影响诊断性玻璃体切除手术诊断准确率的因素

1. 术前全身或眼局部用药：糖皮质激素、免疫抑制剂、抗生素、抗病毒药物等。

2. 标本处理方式不当或污染：标本量少，取材位置不当，送检时间过长且未使用固定液，运输过程中标本泄露等。

3. 临床医师要求的检测项目不全。

4. 既往曾行玻璃体切除术。

5. 实验室检测技术限制。

## 五、诊断性玻璃体切除手术的并发症

虽然诊断性玻璃体切除手术逐步向着更微创、更安全、更准确的方向发展，但仍可能出现并发症，包括视网膜裂孔、视网膜脱离、出血（玻璃体腔、视网膜下或脉络膜）、增生性玻璃体视网膜病变、并发性白内障、一过性高眼压、肿瘤播散等。视网膜脱离更多见于术中进行视网膜脉络膜活体检查者，或原发病存在视网膜脱离风险者，如病毒性视网膜炎。此外，诊断性玻璃体切除手术有导致医源性肿瘤播散风险，因此在决策手术前应做好充分的临床判断，与患者及家属签署知情同意书。诊断性玻璃体切除手术的诊断准确率不是 $100 \%$ ，故患者可能需要再次甚至多次进行手术，但是再次诊断性玻璃体切除手术的诊断准确率可能更低。

## 六、小结

综上所述，尽管诊断性玻璃体切除手术是一种较为安全且有助于确诊眼内病变的方法，但目前仍不建议常规使用，仅推荐用于临床难以确诊的眼内病变，主要包括可疑感染性眼内炎、伪装综合征和眼内占位性病变。儿童更应谨慎行诊断性玻璃体切除手术。
